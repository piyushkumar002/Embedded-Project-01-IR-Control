# Arduino IR Remote Controlled Automation

A simple Arduino project that uses an IR remote to control devices through a relay module.

## Features
- IR Remote Signal Detection
- Relay-Based Device Control
- Beginner-Friendly Circuit
- Easy to Extend for Home Automation

## Hardware Used
- Arduino Uno
- IR Receiver Module
- Relay Module
- TV Remote
- Jumper Wires

## Working
1. IR receiver reads remote button presses.
2. Arduino decodes the received signal.
3. Based on the button pressed, the relay is activated/deactivated.
4. Connected device responds accordingly.
